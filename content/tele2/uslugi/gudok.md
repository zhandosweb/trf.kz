## Описание услуги

Хотите проигрывать мелодию вместо скучных гудков? Для этого у Теле2 есть две услуги: «Гудок» и «Гудок для себя».

С услугой «Гудок» можно поставить мелодию для входящих вызовов. И проигрывать мелодию для абонентов, которые вам звонят.

С услугой «Гудок для себя» можно поставить мелодию для себя. И во время исходящих звонков, вместо гудков для вас будет проигрываться мелодия.

Услуги «Гудок» и «Гудок для себя» две разные услуги. Они тарифицируются по отдельности, можно подключить только одну услугу или обе.

**Стоимость услуги:**

* Для абонентов, которые подключились к услуге до 06.12.2021, стоимость услуги «Гудок» — 7 тенге в сутки, стоимость услуги «Гудок для себя» — 5 тенге в сутки.
* Для абонентов, которые подключили услугу после 06.12.2021, стоимость услуги «Гудок» — 75 тенге в сутки, стоимость услуги «Гудок для себя» — 75 тенге в сутки.

Также подключив услугу «Гудок» или «Гудок для себя», вы получаете ряд опций, о которых рассказываем ниже.

### Какую мелодию можно назначить вместо гудков

После подключения услуги, вы можете назначить мелодию или шутку для одного человека, который вам звонит, для группы контактов (например для семьи или друзей) или для всех звонков.

На официальном сайте услуги перечислены популярные хиты, новинки, а также различные жанры музыки на выбор.

Также можно подключить опцию «Мелодия дня», и каждый день вместо гудков будет проигрываться новая мелодия выбранного жанра. Про эту опцию рассказываем чуть ниже.

Ссылка на официальный сайт услуги: https://gudok.tele2.kz/.

## Как подключить услуги «Гудок» и «Гудок для себя»

**Как подключить «Гудок»:**

* *USSD-команда.* Для подключения услуги «Гудок» наберите на смартфоне USSD-команду `*139*1#` и нажмите клавишу вызова.
* *SMS.* Также услугу можно подключить через отправку SMS. Для этого отправьте сообщение на номер `0550` с текстом `ON`.

После подключения услуги, все звонящие вам абоненты будут слушать стандартную мелодию Теле2 вместо гудков.

**Как подключить «Гудок для себя»:**

* *USSD-команда.* Для подключения услуги наберите USSD-команду: `*139*3#` и нажмите клавишу вызова.
* *SMS.* Для подключения услуги через SMS отправьте сообщение на номер `0550` с текстом `GO`.

После подключения услуги во время исходящих звонков вместо стандартных гудков для вас будет проигрываться стандартная мелодия.

**Как поменять стандартную мелодию:**

После подключения услуг «Гудок» или «Гудок для себя», автоматически проигрывается стандартная мелодия. Если вы хотите поменять стандартную мелодию, есть несколько способов:

* *Через официальный сайт.* Авторизуйтесь на официальном сайте услуги: https://gudok.tele2.kz/. И из каталога мелодий на сайте вы можете выбрать и настроить нужную мелодию вместо стандартного.
* *Через голосовой портал.* Либо же позвоните по короткому номеру 0550 для смены стандартной мелодии.

### Мелодия дня

Также подключив услугу «Гудок» или «Гудок для себя» абоненты получают опцию «Мелодия дня» и «Мелодия дня для себя».

Опция каждый день автоматически будет проигрывать новую мелодию вместо гудков. Каждый месяц будет предоставлена новая уникальная подборка хитов.

**Стоимость услуги:**

* Для абонентов, которые подключали ранее опцию «Гудок дня» до 01.08.2021 числа, опция предоставляется на основе абонентской платы, стоимость — 9.5 тенге в сутки.
* Для абонентов, которые подключили опцию «Мелодия дня» после 01.08.2021 числа, опция предоставляется на бесплатной основе (включена в стоимость услуг «Гудок» и «Гудок для себя»).

Если вы не желаете проигрывать каждый день новую мелодию, опцию можно отключить.

**Как отключить опцию «Мелодия дня»:**

* Для отключения опции наберите USSD-команду: `*139*7#`.
* Либо отправьте SMS на номер `0550` с текстом `456`.

**Как отключить опцию «Мелодия дня для себя»:**

* Чтобы отключить опцию наберите USSD-команду: `*139*8#`.
* Либо отправьте SMS на номер `0550` с текстом `457`.

### Как загрузить свою мелодию

Также абоненты при желании могут загрузить свою мелодию и поставить ее вместо гудков.

Для этого перейдите на официальную страницу Теле2 для загрузки мелодии по ссылке: https://gudok.tele2.kz/ugc и нажмите кнопку «Загрузить мелодию».

Иногда обработка загруженной мелодии может занять несколько часов прежде чем вы сможете ее поставить вместо гудка. Когда мелодия будет загружена, вы получите SMS уведомление.

Всего можно загружать до 10 личных мелодий. В случае превышения количества загруженных мелодии, новые мелодии вы не сможете добавлять, пока не удалите ранее загруженные мелодии.

### Как скопировать мелодию другого абонента

Если во время звонка другому абоненту Теле2 вы услышали мелодию, которая вам понравилась, то вы можете эту мелодию скопировать и поставить у себя.

Для этого во время исходящего вызова пока абонент не поднимет трубку нажмите символ звездочки `*`. И вам придет SMS с названием этой мелодии, который сможете поставить у себя.

Также установленные мелодии абонента можно проверить через сайт. Для этого перейдите на страницу https://gudok.tele2.kz/copy и в специальной форме для поиска мелодии укажите номер собеседника Теле2 и нажмите кнопку «Отправить запрос».

### Установка мелодии через голосовой портал

Если вы не хотите искать на сайте услуги нужные мелодии, вам достаточно воспользоваться специальным голосовым порталом.

Позвоните по короткому номеру 0550 (звонок бесплатный). И вам будет предложен список самых популярных хитов, из которых вы можете выбрать мелодию вместо гудка.

## Как отключить услуги «Гудок» и «Гудок для себя»

**Как отключить услугу «Гудок»:**

* Чтобы отключить услугу «Гудок», наберите на смартфоне USSD-команду `*139*2#` и нажмите клавишу вызова.
* Либо отправьте SMS на номер `0550` с текстом `OFF`.

**Как отключить услугу «Гудок для себя»:**

* Чтобы отключить услугу «Гудок для себя», наберите на смартфоне USSD-команду `*139*4#` и нажмите клавишу вызова.
* Либо отправьте SMS на номер `0550` с текстом `STOP`.
